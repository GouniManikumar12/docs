## Analytics Page

### Key Metrics

| Metric | Description | How to Use |
|--------|-------------|------------|
| **Total Spent** | Sum of CPX + CPC + CPA | Monitor overall budget consumption |
| **Total Queries** | Number of exposures | Track reach and visibility |
| **Total Clicks** | Number of clicks | Measure engagement |
| **Total Conversions** | Number of conversions | Track revenue generation |
| **CTR** | Click-through rate | Optimize ad copy and positioning |
| **CVR** | Conversion rate | Improve landing pages |
| **Cost Per Conversion** | Total Spent / Conversions | Calculate ROI |

### Performance Charts

<AccordionGroup>
  <Accordion title="Spend Over Time">

    **Use cases**:
    - Identify spending spikes
    - Track budget pacing
    - Forecast future spend
    - Compare periods
  </Accordion>

  <Accordion title="Engagement Funnel">

## Queries Page

### Query-Level Data

View detailed information for every exposure:

| Column | Description | Actionable Insights |
|--------|-------------|---------------------|
| **Query** | User's search query | Understand user intent |
| **Product** | Recommended product | See what's being shown |
| **CRS** | Contextual Relevance Score | Identify poor matches |
| **Status** | Exposure/Click/Conversion | Track engagement stage |
| **Cost** | CPX/CPC/CPA amount | Monitor spending |
| **Timestamp** | When query occurred | Analyze timing patterns |
| **Agent** | AI platform source | Track channel performance |

## Offers Page

### Offer Performance

Track each offer's key metrics:

| Metric | Description | Optimization Tip |
|--------|-------------|------------------|
| **Budget Allocated** | Total budget set | Increase for winners |
| **Budget Spent** | Amount consumed | Monitor burn rate |
| **Budget Remaining** | Available budget | Refill before depletion |
| **Exposures** | Times shown | Track visibility |
| **Clicks** | Click count | Measure engagement |
| **Conversions** | Conversion count | Track revenue |
| **CTR** | Click-through rate | Optimize for 5%+ |
| **CVR** | Conversion rate | Optimize for 10%+ |
| **Avg CRS** | Average relevance score | Aim for 70+ |

### Offer Status

<CardGroup cols={3}>
  <Card title="Active" icon="circle-check">

  </Card>
  
  <Card title="Inactive" icon="circle-pause">

### Offer Actions

- **Edit**: Modify offer details (title, description, reward note)
- **Pause**: Temporarily stop offer (preserves budget)
- **Cancel**: Permanently stop offer (releases budget)
- **Duplicate**: Create similar offer with same settings

| **Total Spent** | Cumulative spend across offers | Monitor ROI |

### Product Optimization

<AccordionGroup>
  <Accordion title="Low CRS (< 60)">
    **Problem**: Poor query matching
    
    **Solutions**:
    - Rewrite description with more keywords
    - Add relevant categories
    - Include target audience details
    - Add use cases and benefits
  </Accordion>

  <Accordion title="High CRS, Low CTR">
    **Problem**: Good matches, but users aren't clicking
    
    **Solutions**:
    - Improve offer title/description
    - Increase reward note appeal
    - Test different positioning
    - Add social proof
  </Accordion>

  <Accordion title="High CTR, Low CVR">
    **Problem**: Users click but don't convert
    
    **Solutions**:
    - Optimize landing page
    - Ensure message match
    - Reduce friction in signup
    - Test different CTAs
  </Accordion>
</AccordionGroup>

## Wallet Page

### Budget Overview

| Section | What It Shows | Use Case |
|---------|---------------|----------|
| **Current Balance** | Available funds | Monitor before running out |
| **Total Deposited** | Lifetime deposits | Track total investment |
| **Total Spent** | Lifetime spending | Calculate overall ROI |
| **Locked Budget** | Budget allocated to active offers | Free by cancelling offers |

### Transaction History

View all wallet transactions:

- **Deposits**: Stripe payments and promotional credits
- **Offer Allocations**: Budget locked for new offers
- **Offer Cancellations**: Budget released from cancelled offers
- **CPX Charges**: Exposure costs
- **CPC Charges**: Click costs
- **CPA Charges**: Conversion costs

## Key Performance Indicators (KPIs)

### Essential Metrics to Track

| KPI | Target | Formula |
|-----|--------|---------|
| **CTR** | 3-5%+ | (Clicks / Exposures) × 100 |
| **CVR** | 10-20%+ | (Conversions / Clicks) × 100 |
| **CPA** | < Customer LTV | Total Spent / Conversions |
| **ROI** | 3x-5x+ | Revenue / Total Spent |
| **Avg CRS** | 70+ | Sum of CRS / Total Queries |

### Benchmarks by Industry

| Industry | Typical CTR | Typical CVR | Typical CPA |
|----------|-------------|-------------|-------------|
| **SaaS** | 4-6% | 15-25% | $10-$50 |
| **E-commerce** | 3-5% | 10-15% | $5-$20 |
| **B2B** | 2-4% | 20-30% | $50-$200 |
| **Enterprise** | 1-3% | 25-40% | $200-$1000 |

## Common Questions

<AccordionGroup>
  <Accordion title="How often is data updated?">
    Real-time. All metrics update immediately when events occur.
  </Accordion>
  
  <Accordion title="Can I see which AI platforms are sending traffic?">
    Yes, the Queries page shows the agent (AI platform) for each query.
  </Accordion>
  
  <Accordion title="How do I calculate ROI?">
    ROI = (Revenue - Total Spent) / Total Spent × 100
    
    Example: $1000 revenue, $200 spent = 400% ROI
  </Accordion>
  
  <Accordion title="What's a good conversion rate?">
    Depends on industry, but generally:
    - SaaS: 15-25%
    - E-commerce: 10-15%
    - B2B: 20-30%
  </Accordion>
</AccordionGroup>
