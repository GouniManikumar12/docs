## Our SDKs

AdMesh provides two main SDKs for easy integration:

**admesh-ui-sdk** (React Frontend)
```bash
npm install admesh-ui-sdk
```

**@admesh/weave-node** (Node/Express.js Backend)
```bash
npm install @admesh/weave-node
```

Both SDKs handle all the complexity: session management, recommendation fetching, rendering, tracking, and transparency labels.
