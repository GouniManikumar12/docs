## Quick Example

```bash
# Simple API call
curl -X POST "https://api.useadmesh.com/recommend" \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"query": "best CRM for startups", "format": "auto"}'
```
