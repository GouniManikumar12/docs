## ðŸš€ Quick Start

The fastest way to integrate AdMesh is with the **Provider Pattern** - just **3 lines of code**!

### Step 1: Install
```bash
npm install admesh-ui-sdk@latest
```

### Step 2: Wrap with AdMeshProvider
```tsx
import { AdMeshProvider } from 'admesh-ui-sdk';

<AdMeshProvider apiKey={apiKey} sessionId={sessionId}>
  <YourApp />
</AdMeshProvider>
```

### Step 3: Choose Your Format Component

**Option A: Citation Format (Separate Panel)**
```tsx
import { AdMeshRecommendations } from 'admesh-ui-sdk';

<AdMeshRecommendations messages={messages} format="citation" />
```

**Option B: Product Format (Separate Panel)**
```tsx
import { AdMeshRecommendations } from 'admesh-ui-sdk';

<AdMeshRecommendations messages={messages} format="product" />
```

**Option C: Weave Ad Format (Embedded in LLM Response)**
```tsx
import { WeaveAdFormatContainer, AdMeshRecommendations } from 'admesh-ui-sdk';

<WeaveAdFormatContainer
  messageId={message.id}
  fallbackUI={<AdMeshRecommendations messages={[message]} format="citation" />}
>
  {message.content}
</WeaveAdFormatContainer>
```

---

## Which Format Should I Use?

| Format | Use If | Component |
|--------|--------|-----------|
| **Citation** | You want inline citations in a separate panel | `AdMeshRecommendations` |
| **Product** | You want product cards in a separate panel | `AdMeshRecommendations` |
| **Weave** | You embed links directly in LLM responses | `WeaveAdFormatContainer` |
