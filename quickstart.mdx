## Step 1: Get Your API Key

Sign up at [useadmesh.com](https://useadmesh.com) to get your free API key.

## Step 2: Make Your First API Call

### Basic cURL Example

```bash
curl -X POST "https://api.useadmesh.com/agent/recommend" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "best CRM for startups",
    "agent_id": "your_agent_id"
  }'
```

**Response:**
```json
{
  "recommendations": [
    {
      "title": "HubSpot CRM",
      "reason": "Perfect for startups with excellent free tier",
      "admesh_link": "https://useadmesh.com/track?ad_id=hubspot-123",
      "pricing": "Free tier available, paid plans from $45/month"
    }
  ]
}
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

type RecommendationRequest struct {
    Query   string `json:"query"`
    AgentID string `json:"agent_id"`
}

func getRecommendations(query, agentID string) ([]interface{}, error) {
    reqBody := RecommendationRequest{
        Query:   query,
        AgentID: agentID,
    }

    jsonData, _ := json.Marshal(reqBody)

    req, _ := http.NewRequest("POST", "https://api.useadmesh.com/agent/recommend", bytes.NewBuffer(jsonData))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        return nil, err
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)

    var result map[string]interface{}
    json.Unmarshal(body, &result)

    return result["recommendations"].([]interface{}), nil
}

// Usage
func main() {
    recommendations, _ := getRecommendations("best CRM for startups", "your_agent_id")
    fmt.Println(recommendations)
}
```

```ruby Ruby
require 'net/http'
require 'json'

def get_recommendations(query, agent_id)
  uri = URI('https://api.useadmesh.com/agent/recommend')

  http = Net::HTTP.new(uri.host, uri.port)
  http.use_ssl = true

  request = Net::HTTP::Post.new(uri)
  request['Authorization'] = 'Bearer YOUR_API_KEY'
  request['Content-Type'] = 'application/json'
  request.body = {
    query: query,
    agent_id: agent_id
  }.to_json

  response = http.request(request)
  result = JSON.parse(response.body)

  result['recommendations']
end

# Usage
recommendations = get_recommendations('best CRM for startups', 'your_agent_id')
```

  const [recommendations, setRecommendations] = useState([]);

  const fetchRecommendations = async (query) => {

  <Accordion title="Understanding the Response">

Each recommendation includes:

- **`title`** - Product/service name
- **`reason`** - Why it's recommended for this query
- **`admesh_link`** - Tracking URL (use this for clicks)
- **`pricing`** - Cost information
- **`features`** - Key features list
- **`trial_days`** - Free trial period
