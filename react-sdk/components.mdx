# React Components

The AdMesh React SDK provides pre-built components for displaying recommendations with automatic tracking and styling.

## RecommendationCard

Display a single recommendation with image, title, description, and CTA button.

### Usage

```tsx
import { RecommendationCard } from '@admesh/react-sdk';

function MyComponent() {
  const recommendation = {
    id: '123',
    title: 'MacBook Pro 16"',
    description: 'Powerful laptop for professionals',
    image_url: 'https://example.com/image.jpg',
    url: 'https://example.com/product',
    price: 2499,
    rating: 4.8
  };

  return (
    <RecommendationCard
      recommendation={recommendation}
      onClick={() => console.log('Clicked')}
      onTrack={(event) => console.log('Tracked:', event)}
    />
  );
}
```

### Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `recommendation` | Recommendation | Yes | Recommendation object |
| `onClick` | function | No | Click handler |
| `onTrack` | function | No | Tracking event handler |
| `showRating` | boolean | No | Show rating (default: true) |
| `showPrice` | boolean | No | Show price (default: true) |
| `className` | string | No | Custom CSS class |

## RecommendationGrid

Display multiple recommendations in a responsive grid layout.

### Usage

```tsx
import { RecommendationGrid } from '@admesh/react-sdk';

function MyComponent() {
  const recommendations = [
    { id: '1', title: 'Product 1', ... },
    { id: '2', title: 'Product 2', ... },
    { id: '3', title: 'Product 3', ... }
  ];

  return (
    <RecommendationGrid
      recommendations={recommendations}
      columns={3}
      loading={false}
      error={null}
      onCardClick={(rec) => console.log('Clicked:', rec)}
    />
  );
}
```

### Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `recommendations` | Recommendation[] | Yes | Array of recommendations |
| `columns` | number | No | Grid columns (default: 3) |
| `loading` | boolean | No | Show loading state |
| `error` | Error \| null | No | Show error state |
| `onCardClick` | function | No | Card click handler |
| `onTrack` | function | No | Tracking event handler |
| `gap` | string | No | Grid gap (default: '1rem') |
| `className` | string | No | Custom CSS class |

## AdMeshProvider

Context provider for configuring the SDK globally.

### Usage

```tsx
import { AdMeshProvider } from '@admesh/react-sdk';

function App() {
  return (
    <AdMeshProvider
      config={{
        apiKey: 'admesh_prod_your_key',
        apiBaseUrl: 'https://api.admesh.com',
        debug: true,
        viewabilityConfig: {
          threshold: 50,
          duration: 1000
        }
      }}
    >
      <YourApp />
    </AdMeshProvider>
  );
}
```

### Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `config` | AdMeshSDKConfig | Yes | SDK configuration |
| `children` | ReactNode | Yes | Child components |

## Styling

All components support customization through CSS classes and inline styles.

### Dark Mode

Components automatically detect dark mode using `prefers-color-scheme`:

```tsx
<RecommendationCard
  recommendation={rec}
  className="dark:bg-slate-900 dark:text-white"
/>
```

### Responsive Design

Components are mobile-responsive by default:

```tsx
<RecommendationGrid
  recommendations={recs}
  columns={1}  // Mobile
  // Automatically adjusts to 2-3 columns on larger screens
/>
```

## Customization

### Custom Styling

```tsx
<RecommendationCard
  recommendation={rec}
  className="custom-card"
/>
```

```css
.custom-card {
  border-radius: 12px;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
  padding: 1.5rem;
}
```

### Custom Rendering

For advanced use cases, use hooks directly:

```tsx
import { useRecommendations, useViewabilityTracker } from '@admesh/react-sdk';

function CustomRecommendation() {
  const { ref, isVisible } = useViewabilityTracker();
  const { recommendations } = useRecommendations({ query: 'laptop' });

  return (
    <div ref={ref}>
      {/* Your custom rendering */}
    </div>
  );
}
```
