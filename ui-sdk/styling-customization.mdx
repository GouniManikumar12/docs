## Overview

The AdMesh UI SDK includes a simple, framework-agnostic styling system. It ensures your components look clean and consistent—no matter where they’re embedded (Next.js, React, Vite, or platforms like Perplexica).

## Quick Start

By default, the SDK injects its own scoped styles automatically. No CSS import is needed.

```tsx
import { AdMeshSDK } from 'admesh-ui-sdk';

const admesh = new AdMeshSDK({ apiKey: process.env.REACT_APP_ADMESH_API_KEY });

await admesh.showRecommendations({
  query: 'best CRM software',
  containerId: 'admesh-recommendations',
  format: 'auto'
});
```

## Custom Theming

You can modify the appearance globally with the theming API.

```tsx
import { AdMeshSDK, mergeTheme, injectCustomTheme } from 'admesh-ui-sdk';

const theme = mergeTheme({
  mode: 'dark',
  primaryColor: '#3b82f6',
  accentColor: '#ffffff',
  borderRadius: '0.5rem'
});

injectCustomTheme(theme);

const admesh = new AdMeshSDK({ apiKey: process.env.REACT_APP_ADMESH_API_KEY });

await admesh.showRecommendations({
  query: 'best CRM software',
  containerId: 'admesh-recommendations',
  format: 'auto'
});
```

## Theming Functions

| Function | Description |
|-----------|-------------|
| `mergeTheme(config?)` | Merge custom settings with default theme |
| `injectCustomTheme(theme)` | Apply theme globally |
| `generateThemeVariables(theme)` | Convert theme to CSS variables |
| `createThemeProvider()` | Manage theme state dynamically |

## CSS Variables

You can override AdMesh default styling with standard CSS variables.

```css
:root {
  --admesh-primary-color: #3b82f6;
  --admesh-border-radius: 10px;
  --admesh-font-family: 'Inter', sans-serif;
  --admesh-shadow-md: 0 4px 12px rgba(0, 0, 0, 0.08);
}
```

## Integration Tips

### Next.js + Tailwind

AdMesh styles are isolated from Tailwind classes, so you can safely use both.

```tsx
import { AdMeshSDK, mergeTheme, injectCustomTheme } from 'admesh-ui-sdk';

export default function Recommendations() {
  const theme = mergeTheme({ mode: 'light' });
  injectCustomTheme(theme);

  const admesh = new AdMeshSDK({ apiKey: process.env.NEXT_PUBLIC_ADMESH_API_KEY });

  return <div id="admesh-recommendations" className="p-4 bg-white rounded-lg" />;
}
```

### React + Bootstrap

```tsx
import { AdMeshSDK, mergeTheme, injectCustomTheme } from 'admesh-ui-sdk';

export default function App() {
  const theme = mergeTheme({ mode: 'light' });
  injectCustomTheme(theme);

  const admesh = new AdMeshSDK({ apiKey: process.env.REACT_APP_ADMESH_API_KEY });

  return <div className="container"><div id="admesh-recommendations" /></div>;
}
```
