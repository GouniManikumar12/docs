## Overview

The AdMesh UI SDK includes a comprehensive platform-agnostic styling system that ensures consistent rendering across all platforms (Perplexica, custom React apps, Next.js, etc.) without being affected by host platform CSS frameworks like Tailwind, Bootstrap, Material-UI, or Chakra UI.

## How Style Isolation Works

The styling system uses three layers of isolation:

1. **CSS Reset** - Normalizes all styles within `.admesh-component` boundary
2. **Scoped Styles** - All AdMesh components use `.admesh-component` class prefix
3. **CSS-in-JS** - Styles are injected dynamically and isolated from global CSS

```tsx
// All AdMesh components automatically use the .admesh-component class
<div className="admesh-component">
  {/* Component content is isolated from host platform CSS */}
</div>
```

## Quick Start

### Default Usage (No Configuration)

Components work out-of-the-box with default styling:

```tsx
import { AdMeshSDK } from 'admesh-ui-sdk';

const admesh = new AdMeshSDK({
  apiKey: process.env.REACT_APP_ADMESH_API_KEY
});

// Render with default styling
await admesh.showRecommendations({
  query: 'best CRM software',
  containerId: 'admesh-recommendations',
  format: 'auto'
});
```

### With Custom Theme

```tsx
import {
  AdMeshSDK,
  mergeTheme,
  injectCustomTheme
} from 'admesh-ui-sdk';

// Create custom theme
const theme = mergeTheme({
  mode: 'dark',
  primaryColor: '#3b82f6',
  accentColor: '#ffffff',
  borderRadius: '0.5rem'
});

// Inject globally (do this once at app startup)
injectCustomTheme(theme);

// Initialize SDK
const admesh = new AdMeshSDK({
  apiKey: process.env.REACT_APP_ADMESH_API_KEY
});

// Render with theme (automatically applied)
await admesh.showRecommendations({
  query: 'best CRM software',
  containerId: 'admesh-recommendations',
  format: 'auto'
});
```

## Theming API

### mergeTheme()

Merges user theme configuration with default theme:

```tsx
import { mergeTheme } from 'admesh-ui-sdk';

// Light theme (default)
const lightTheme = mergeTheme();

// Dark theme
const darkTheme = mergeTheme({ mode: 'dark' });

// Custom theme
const customTheme = mergeTheme({
  mode: 'light',
  primaryColor: '#6366f1',
  accentColor: '#000000',
  backgroundColor: '#ffffff',
  textColor: '#111827',
  borderColor: '#e5e7eb',
  borderRadius: '0.75rem'
});
```

### injectCustomTheme()

Injects custom theme CSS into the document:

```tsx
import { injectCustomTheme, mergeTheme } from 'admesh-ui-sdk';

const theme = mergeTheme({ mode: 'dark' });
injectCustomTheme(theme); // Inject globally
```

### Available Theme Functions

| Function | Purpose |
|----------|---------|
| `mergeTheme(config?)` | Merge user config with defaults |
| `injectCustomTheme(theme)` | Inject theme CSS globally |
| `generateThemeVariables(theme)` | Generate CSS custom properties |
| `getCurrentThemeMode(mode?)` | Get current theme mode |
| `createThemeProvider(config?)` | Create theme provider for state management |
| `detectCSSFramework()` | Detect host platform CSS frameworks |
| `injectAdMeshStyles()` | Inject AdMesh CSS reset and core styles |
| `areAdMeshStylesInjected()` | Check if styles are injected |

## CSS Variables

Customize AdMesh components using CSS variables:

```css
:root {
  /* Colors */
  --admesh-primary-color: #007bff;
  --admesh-secondary-color: #6c757d;
  --admesh-success-color: #28a745;
  
  /* Background colors */
  --admesh-bg-primary: #ffffff;
  --admesh-bg-secondary: #f8f9fa;
  
  /* Text colors */
  --admesh-text-primary: #212529;
  --admesh-text-secondary: #6c757d;
  
  /* Spacing */
  --admesh-spacing-md: 16px;
  --admesh-spacing-lg: 24px;
  
  /* Border radius */
  --admesh-border-radius-md: 8px;
  
  /* Shadows */

  /* Typography */

  --admesh-font-size-md: 1rem;
}
```

## Platform-Specific Integration

### Next.js with Tailwind

```tsx
import { AdMeshSDK, mergeTheme, injectCustomTheme } from 'admesh-ui-sdk';
import { useEffect } from 'react';

export const RecommendationsComponent = () => {
  useEffect(() => {
    // Inject theme once at app startup
    const theme = mergeTheme({ mode: 'dark' });
    injectCustomTheme(theme);
  }, []);

  const admesh = new AdMeshSDK({
    apiKey: process.env.NEXT_PUBLIC_ADMESH_API_KEY
  });

  return (
    <div className="bg-white p-4">
      {/* Tailwind styles won't affect AdMesh */}
      <div id="admesh-recommendations" />
    </div>
  );
};
```

### Custom React App with Bootstrap

```tsx
import { AdMeshSDK, mergeTheme, injectCustomTheme, detectCSSFramework } from 'admesh-ui-sdk';
import { useEffect } from 'react';

export const BootstrapApp = () => {
  useEffect(() => {
    // Detect Bootstrap and apply appropriate theme
    const frameworks = detectCSSFramework();
    const theme = mergeTheme({
      mode: frameworks.includes('bootstrap') ? 'light' : 'dark'
    });
    injectCustomTheme(theme);
  }, []);

  const admesh = new AdMeshSDK({
    apiKey: process.env.REACT_APP_ADMESH_API_KEY
  });

  return (
    <div className="container">
      <div id="admesh-recommendations" />
    </div>
  );
};
```

## Advanced Customization

### Runtime Theme Switching

```tsx
import { createThemeProvider } from 'admesh-ui-sdk';
import { useState } from 'react';

const provider = createThemeProvider({ mode: 'light' });

export const ThemeSwitcher = () => {
  const [isDark, setIsDark] = useState(false);

  const toggleTheme = () => {
    const newMode = isDark ? 'light' : 'dark';
    provider.updateTheme({ mode: newMode });
    setIsDark(!isDark);
  };

  return (
    <button onClick={toggleTheme}>
      Toggle {isDark ? 'Light' : 'Dark'} Mode
    </button>
  );
};
```

```css

  --admesh-accent: #ffffff;

}

.admesh-component .admesh-card:hover {
  transform: translateY(-4px);
}
```
